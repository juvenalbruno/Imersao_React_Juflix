<h3>ℹ️ Como usar</h3>

    # Clone esse repositório
    $ git clone https://github.com/juvenalbruno/Imersao_React_Juflix
    
    # Instalar dependências do projeto
    $ npm install
    
    # Executar o comando para rodar o projeto
    $ npm start
